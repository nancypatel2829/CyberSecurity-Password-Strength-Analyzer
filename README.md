# Cyber Security Password Strength Analyzer

## Project Description

This project is a simple Python application that checks the strength of a password. It helps users create secure passwords by checking the password length and displaying whether the password is weak or strong.

## Features

* User-friendly interface
* Checks password length
* Displays password strength
* Written in Python
* Beginner-friendly cybersecurity project

## Technologies Used

* Python 3
* Visual Studio Code

## How to Run

1. Open the project folder.
2. Open Terminal.
3. Run:

python password_strength.py

4. Enter a password.
5. View the password strength.
